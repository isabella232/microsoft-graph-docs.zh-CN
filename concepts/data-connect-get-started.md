# <a name="get-started-with-microsoft-graph-data-connect-preview"></a>开始使用 Microsoft Graph 数据连接 （预览）

您可以使用数据连接的 Microsoft Graph 之前，Office 365 管理员必须执行两个操作，这两种启用与控件数据移动到特权访问管理 (PAM) 管理的功能。 

1. 授予许可选择 Microsoft Graph 通过 Microsoft 365 管理门户中，在**服务和外接程序**页上的数据连接。 这将使数据对 Microsoft Azure （即，保留完全控制数据，但允许 Azure 资源访问） 的移动请求。 除非为特定管道审批提供更高版本，则会不传输任何数据。
2. 设置 Office 365 订阅中的审批者组。 请确保审批者组不为空。 组中的用户可以审核数据的移动请求。

有关详细的分步说明，请参阅[Microsoft Graph 数据连接培训模块](https://github.com/microsoftgraph/msgraph-training-dataconnect/blob/master/Lab.md)。

## <a name="next-steps"></a>后续步骤
