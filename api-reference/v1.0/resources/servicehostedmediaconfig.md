# <a name="servicehostedmediaconfig-resource-type"></a>serviceHostedMediaConfig 资源类型

远程承载的媒体。 这是从[mediaConfig](mediaconfig.md)继承的。

## <a name="properties"></a>属性

| 属性                    | 类型                                                        | 说明                                       |
| :-------------------------- | :---------------------------------------------------------- | :-------------------------------------------------|
| preFetchMedia               | [mediaInfo](mediainfo.md)集合                        | 预回迁的介质列表。                   |
