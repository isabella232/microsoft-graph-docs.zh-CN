# <a name="devicemanagementexchangeaccessstate-enum-type"></a>deviceManagementExchangeAccessState 枚举类型

> **注意：** 使用 Microsoft Graph API 配置 Intune 控件和策略仍需要客户[正确许可](https://go.microsoft.com/fwlink/?linkid=839381) Intune 服务。

设备 Exchange 访问状态。
## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|none|0|没有从 Exchange 中发现访问状态|
|unknown|1|到 Exchange 的设备访问状态是未知|
|allowed|2|设备有权访问 Exchange|
|blocked|3|设备在 Exchange 中被阻止|
|quarantined|4|设备在 Exchange 中被隔离|
