# <a name="ratingcanadatelevisiontype-enum-type"></a>ratingCanadaTelevisionType 枚举类型

> **注意：** 使用 Microsoft Graph API 配置 Intune 控件和策略仍需要客户[正确许可](https://go.microsoft.com/fwlink/?linkid=839381) Intune 服务。

在加拿大的电视内容评级标签
## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|allAllowed|0|默认值，允许所有电视都显示内容|
|allBlocked|1|不允许任何电视显示内容|
|children|2|C 分类适合 2 到 7 岁的儿童|
|childrenAbove8|3|C8 分类适合 8 岁以上的儿童|
|general|4|G 分类适合普通观众|
|parentalGuidance|5|PG，父母指导|
|agesAbove14|6|14+ 分类适用于 14 岁及以上的观众|
|agesAbove18|7|18+ 分类适用于 18 岁及以上的观众|
