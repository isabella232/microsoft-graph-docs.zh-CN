# <a name="ivr-scenarios-in-calls"></a>调用中的 IVR 方案

以下是 Microsoft Graph 中的调用 Api 支持的交互语音响应（IVR）方案：

- 在客户服务代理的队列中进行呼叫时，播放音频提示（例如）。
- 录制响应-例如，录制呼叫者的音频（通常是在听到选项提示后）。
- 订阅声音-例如，当您想要了解呼叫者选择的 DTMF 铃声时，通常会听到音频提示。
