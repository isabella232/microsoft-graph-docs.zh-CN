美国的电视内容评级标签

## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|allAllowed|0|默认值, 允许所有电视节目内容|
|allBlocked|1|不允许任何电视显示内容|
|childrenAll|2 |TV-Y, 所有子项|
|childrenAbove7|3 |电视-Y7, 儿童年龄7及更高版本|
|概要|4 |TV-G, 适用于所有年龄|
|parentalGuidance|5 |TV PG、家长指导|
|childrenAbove14|6 |电视-14, 儿童年龄14及更高版本|
|成人|7 |仅限成人的电视-MA|
