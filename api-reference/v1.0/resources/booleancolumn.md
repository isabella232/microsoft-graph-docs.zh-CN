# <a name="booleancolumn-resource-type"></a>BooleanColumn 资源类型

[columnDefinition](columndefinition.md) 资源上的 **booleanColumn** 指示该列包含一个布尔值。

下面是 **booleanColumn** 资源的 JSON 表示形式。

}

## <a name="properties"></a>属性
