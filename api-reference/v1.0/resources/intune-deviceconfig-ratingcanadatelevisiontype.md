## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|allAllowed|0|默认值，允许所有电视节目内容|

|childrenAbove8|第三章|C8 分类适用于仅有8岁以上的儿童|

|parentalGuidance|5 |PG、家长指导|

|agesAbove18|7 |18个以上的分类适用于18岁及更早的查看者|
