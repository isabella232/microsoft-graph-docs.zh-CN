# <a name="windowsspotlightenablementsettings-enum-type"></a>windowsSpotlightEnablementSettings 枚举类型

> **注意：** 使用 Microsoft Graph API 配置 Intune 控件和策略仍需要客户[正确许可](https://go.microsoft.com/fwlink/?linkid=839381) Intune 服务。

允许 IT 管理员为 MDM 控制设备预设默认搜索引擎。
## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|notConfigured|0|锁屏上的热点未配置|
|disabled|1|在锁屏上禁用 Windows 热点|
|enabled|2|在锁屏上启用 Windows 热点|
