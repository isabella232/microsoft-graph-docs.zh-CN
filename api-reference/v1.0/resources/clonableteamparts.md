# <a name="clonableteamparts-enum-type"></a>clonableTeamParts 枚举类型



描述应克隆的[工作组](../resources/team.md)的一部分。 

## <a name="members"></a>成员

| 成员 | 值| 说明 |
|:---------------|:--------|:----------|
|apps|1|将复制已安装的应用程序的列表。|
|选项卡|2|将复制通道中的选项卡。|
|settings|4|将复制的团队，以及关键组设置中的所有设置。|
|通道|8|将复制的通道结构 （但不是在进入频道的消息）。|
|members|16|将复制的成员和所有者的团队。|
