# <a name="phone-resource-type"></a>phone 资源类型

表示一个电话号码。


## <a name="properties"></a>属性
| 属性     | 类型   |说明|
|:---------------|:--------|:----------|
|number|string|电话号码。|
|type|String|电话号码的类型。可能的值是：`home`、`business`、`mobile`、`other`、`assistant`、`homeFax`、`businessFax`、`otherFax`、`pager`、`radio`。|
