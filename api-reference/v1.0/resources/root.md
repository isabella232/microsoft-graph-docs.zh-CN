# <a name="root-resource-type"></a>Root 资源类型

**Root** Facet 指示对象是它的层次结构中最顶层的对象。
如果存在 Facet 值（非 null），则表明对象是根。
如果值为 null（或缺失），则表明对象不是根。

**注意**：尽管此 Facet 暂为空，但在今后推出的 API 版本中将在此 Facet 中填充其他属性。
