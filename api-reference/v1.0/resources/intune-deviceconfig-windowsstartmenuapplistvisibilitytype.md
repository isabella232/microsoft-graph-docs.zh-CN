类型的列表开始菜单应用程序的可见性。
## <a name="members"></a>成员
|成员|值|Description|
|:---|:---|:---|
|用户定制|0|用户定义。 默认值。|
|折叠|1|折叠开始菜单上的应用程序列表中。|
|删除|2|完全从开始菜单中删除应用程序列表。|
|disableSettingsApp|4|禁用设置应用程序中的相应切换 （折叠或删除）。|
