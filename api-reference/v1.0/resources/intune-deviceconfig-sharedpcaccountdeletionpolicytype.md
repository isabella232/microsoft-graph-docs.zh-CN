## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|立即|0|立即删除。|
|diskSpaceThreshold|1|删除在磁盘空间阈值。|
|diskSpaceThresholdOrInactiveThreshold|2|删除在磁盘空间阈值或非活动状态的阈值。|
