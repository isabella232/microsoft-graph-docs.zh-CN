# <a name="website-resource-type"></a>website 资源类型

表示一个网站。


## <a name="properties"></a>属性
| 属性     | 类型   |说明|
|:---------------|:--------|:----------|
|type|websiteType| 可能的值为： `other`， `home`， `work`， `blog`， `profile`。|
|address|string|网站的 URL。|
|displayName|string|网站的显示名称。|
