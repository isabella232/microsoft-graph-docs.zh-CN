# <a name="windowsarchitecture-enum-type"></a>windowsArchitecture 枚举类型

> **注意：** 使用 Microsoft Graph API 配置 Intune 控件和策略仍需要客户[正确许可](https://go.microsoft.com/fwlink/?linkid=839381) Intune 服务。

包含 Windows 体系结构的属性。
## <a name="windowsarchitecture-flags"></a>windowsArchitecture 标志
|成员|值|说明|
|:---|:---|:---|
|none|0|无设置标志。|
|x86|1|是否支持 X86 Windows 体系结构类型。|
|x64|2|是否支持 X64 Windows 体系结构类型。|
|arm|4|是否支持 Arm Windows 体系结构类型。|
|neutral|8|是否支持 Neutral Windows 体系结构类型。|
