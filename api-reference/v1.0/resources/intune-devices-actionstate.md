设备上操作的状态

## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|无|0|操作状态无效|
|决|1|操作挂起|
|取消|2 |操作已被取消。|
|工作|3 |操作处于活动状态。|
|done|4 |操作已完成, 无错误。|
|未能|5 |操作失败|
|notSupported|6 |操作不受支持。|
