# <a name="applisttype-enum-type"></a>appListType 枚举类型

> **注意：** 使用 Microsoft Graph API 配置 Intune 控件和策略仍需要客户[正确许可](https://go.microsoft.com/fwlink/?linkid=839381) Intune 服务。

合规性应用程序列表的可能值。
## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|none|0|默认值，没有用途。|
|appsInListCompliant|1|列表代表将被视为合规的应用程序 （仅在列表上的应用程序视为合规）。|
|appsNotInListCompliant|2|列表代表将被视为不合规的应用程序 （所有应用程序都合规，列表上的应用程序除外）。|
