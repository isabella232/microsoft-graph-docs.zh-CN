监视文件活动的可能值。

## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|定制|0|用户定义, 默认值, 无意向。|
|启用|1|禁用监控文件活动。|
|monitorAllFiles|双面|监视所有文件。|
|monitorIncomingFilesOnly|第三章| 仅监视传入的文件。|
|monitorOutgoingFilesOnly|4|仅监视传出文件。|
