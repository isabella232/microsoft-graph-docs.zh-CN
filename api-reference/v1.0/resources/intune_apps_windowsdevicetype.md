# <a name="windowsdevicetype-enum-type"></a>windowsDeviceType 枚举类型

> **注意：** 使用 Microsoft Graph API 配置 Intune 控件和策略仍需要客户[正确许可](https://go.microsoft.com/fwlink/?linkid=839381) Intune 服务。

包含 Windows 设备类型的属性。
## <a name="windowsdevicetype-flags"></a>windowsDeviceType 标志
|Member|值|说明|
|:---|:---|:---|
|none|0|无设置标志。|
|desktop|1|是否支持桌面 Windows 设备类型。|
|mobile|2|是否支持 Mobile Windows 设备类型。|
|holographic|4|是否支持全息 Windows 设备类型。|
|team|8|是否支持团队 Windows 设备类型。|
