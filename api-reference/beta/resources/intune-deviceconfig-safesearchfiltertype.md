指定需要何种级别的安全搜索 （筛选成人内容）
## <a name="members"></a>成员
|成员|值|Description|
|:---|:---|:---|
|用户定制|0|用户定义，默认值、 没有用途。|
|严格|1|严格，最高针对成人内容筛选。|
|中等|2|中等筛选成人内容 （有效的搜索结果将不会筛选）。|
