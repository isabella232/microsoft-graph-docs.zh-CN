Windows 更新通知显示选项

## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|notConfigured|0|未配置|
|defaultNotifications|1 |使用默认的 Windows 更新通知。|
|restartWarningsOnly|2 |关闭所有通知，但不包括重新启动警告。|
|disableAllNotifications|3 |关闭所有通知，包括重启警告。|
