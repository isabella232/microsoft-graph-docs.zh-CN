"开始" 菜单应用程序列表可见性的类型。

## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|定制|0|用户定义。 默认值。|
|折叠|1|折叠 "开始" 菜单上的 "应用" 列表。|
|删除|双面|从 "开始" 菜单中完全删除应用列表。|
|disableSettingsApp|4 |禁用 "设置" 应用中的相应切换（折叠或删除）。|
