## <a name="properties"></a>属性
|属性|类型|说明|
|:---|:---|:---|
|id|String|Device management script group 分配实体的键。 此属性是只读的。|
|targetGroupId|String|要作为脚本目标的 Azure Active Directory 组的 Id。|
