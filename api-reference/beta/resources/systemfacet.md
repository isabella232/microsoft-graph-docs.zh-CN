# <a name="system-facet"></a>系统 Facet

命名空间：microsoft.graph

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

**系统** Facet 指示对象由系统为其自己的操作进行管理。
大多数应用应忽略具有系统 Facet 的项。

**注意**：尽管此 Facet 暂为空，但在今后推出的 API 版本中将在此 Facet 中填充其他属性。
