# <a name="freebusyerror-resource-type"></a>freeBusyError 资源类型

 [!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]
 
表示尝试获取用户、通讯组列表或资源可用性的错误信息。

## <a name="properties"></a>属性
| 属性     | 类型   |说明|
|:---------------|:--------|:----------|
|message |String |描述错误。 |
|responseCode |String |对用户、通讯组列表或资源的可用性进行查询的响应代码。 |
