# <a name="clonableteamparts-enum-type"></a>clonableTeamParts 枚举类型



介绍应克隆的[团队](../resources/team.md)部分。 

## <a name="members"></a>成员

| 成员 | 值| 说明 |
|:---------------|:--------|:----------|
|apps|1|复制已安装应用程序的列表。|
|选项卡|2 |复制通道中的选项卡。|
|settings|4 |复制团队中的所有设置以及关键组设置。|
|channels|8 |复制通道结构 (而不是通道中的邮件)。|
|members|16 |复制团队的成员和所有者。|
