# <a name="policybase-resource-type"></a>policyBase 资源类型

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

表示要从其继承的策略类型的抽象基类型。

## <a name="methods"></a>Methods

无

## <a name="properties"></a>属性

| 属性     | 类型        | 说明 |
|:-------------|:------------|:------------|
