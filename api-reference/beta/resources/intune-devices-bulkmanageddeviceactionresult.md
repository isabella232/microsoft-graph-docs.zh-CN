尚未记录
## <a name="properties"></a>属性
|属性|类型|说明|
|:---|:---|:---|
|successfulDeviceIds|String 集合|成功的设备|
|failedDeviceIds|String 集合|失败的设备|
|notFoundDeviceIds|String 集合|未找到设备|
|notSupportedDeviceIds|String 集合|不受支持的设备|
