# <a name="chartfill-resource-type"></a>ChartFill 资源类型

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

表示图表元素的格式填充。


## <a name="methods"></a>方法

| 方法           | 返回类型    |说明|
|:---------------|:--------|:----------|

|[Setsolidcolor](../api/chartfill-setsolidcolor.md)|无|将图表元素的填充格式设置为统一颜色。|
