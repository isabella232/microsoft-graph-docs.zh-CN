# <a name="accessaction-resource-type"></a>accessAction 资源类型

[!INCLUDE [beta-disclaimer](../../includes/beta-disclaimer.md)]

>**注意:** Access 活动记录目前仅适用于 SharePoint 和 OneDrive for business。
