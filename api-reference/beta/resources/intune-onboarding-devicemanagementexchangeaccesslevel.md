> **注意：** 适用于 Intune 的 Microsoft Graph API 需要适用于租户的[活动 Intune 许可证](https://go.microsoft.com/fwlink/?linkid=839381)。

Exchange 中的访问级别。

## <a name="members"></a>成员
|成员|值|说明|
|:---|:---|:---|
|无|0|未在 Exchange 中配置任何设备访问规则。|
|允许|1 |允许设备访问 Exchange。|
|数据|2 |阻止设备访问 Exchange。|
|隔离|3 |隔离 Exchange 中的设备。|
