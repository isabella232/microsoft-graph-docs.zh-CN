# <a name="manage-resource-access-certificates-in-intune"></a>管理 Intune 中的资源访问证书

> **重要说明：** Microsoft Graph 中 /beta 版本下的 API 是预览版，可能会发生变化。 在生产应用程序中不支持使用这些 API。

> **注意：** 使用 Microsoft Graph API 配置 Intune 控件和策略仍需要客户[正确许可](https://www.microsoft.com/en-us/cloud-platform/microsoft-intune-pricing) Intune 服务。

可以导入 Microsoft Graph API 用于 Intune 的 PFX 证书。

以下 Graph 资源可用于在 Intune 中管理注册：

- [用户 PFX 证书](intune-raimportcerts-userpfxcertificate.md)
- [用户应 PFX 用途](intune-raimportcerts-userpfxintendedpurpose.md)
- [用户 PFX 填充方案](intune-raimportcerts-userpfxpaddingscheme.md)
